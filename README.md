# Descrição
Um **repositório simples** para armazenamento e compartilhamento de soluções de questões da OBI feitas por alunos e professores no projeto preparatório para a OBI no **Colégio Pedro Segundo Campus Tijuca II**

# Organização
* Os níveis **Júnior** e **Sênior** são **ignorados**
* As **pastas** são organizadas em **Nível > Ano > Fase > Questão**. Ex. de caminho: *"Nível 1/2019/Fase 1/A idade de Dona Mônica/"*
* Caso a **mesma questão** apareça em provas de **diferentes níveis**, é recomendado colocá-la no lugar do **menor nível** que ela aparece. Ex.: A questão *"A idade de Dona Mônica"* aparece em 2019 nos níveis 1 e 2, portanto é recomendado colocá-la no nível 1, pois é o menor
* **Caso não exista o caminho para uma questão** que você está fazendo, **você mesmo pode criar o caminho** seguindo as instruções acima
* O **nome do arquivo** da sua solução deve ser um identificador seu, como nome, sobrenome, apelido ou nickname. (isso pode mudar para apenas nome e/ou sobrenome)
* Caso tenha **mais de uma solução** para a mesma questão que queira guardar no repositório, nomeie-as como **sequências de filme**. Ex.: *"gumbito.py"*, *"gumbito2.py"*, *"gumbito3.py"*
* *(Eu pensei em deixar um "questão.txt" contendo o link da questão, mas já é bem fácil achar a questão pelo nome, ano e fase. Vocês que sabem se isso vira uma regra de organização ou não)*
